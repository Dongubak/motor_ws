# 속도·가속도 설정 가이드

> **대상 파일:** `src/motor_control/config/motor_params.yaml`

---

## 1. RPM이란?

**RPM (Revolutions Per Minute)** = 모터 축이 1분에 몇 바퀴 도는지를 나타내는 단위입니다.

```
1 RPM = 1분에 1바퀴 회전
```

### 회전이 실제 이동 거리로 바뀌는 원리

모터 1바퀴 = 기계가 일정 거리 이동합니다. 이 거리를 **mm/rev** (1회전당 mm) 라고 합니다.

| 축 | 1회전당 이동 거리 | 비고 |
|----|-----------------|------|
| X축 | 11.999 mm/rev | 볼스크류 피치 기반 |
| Z축 | 5.999 mm/rev  | 볼스크류 피치 기반 |

### RPM → mm/s 변환 공식

```
속도 [mm/s] = RPM × mm/rev ÷ 60
```

**예시 (X축, 30 RPM):**
```
30 RPM × 11.999 mm/rev ÷ 60 = 6.0 mm/s
```

**예시 (Z축, 30 RPM):**
```
30 RPM × 5.999 mm/rev ÷ 60 = 3.0 mm/s
```

### mm/s → RPM 역변환 공식

```
RPM = mm/s × 60 ÷ mm/rev
```

**예시 (X축, 50 mm/s 원할 때):**
```
50 × 60 ÷ 11.999 = 250 RPM
```

**예시 (Z축, 50 mm/s 원할 때):**
```
50 × 60 ÷ 5.999 = 500 RPM
```

## 2. mm/s² 란? (가속도)

**mm/s²** = 속도가 1초 동안 얼마나 빨라지는지를 나타냅니다.

```
가속도 1 mm/s² = 1초가 지날 때마다 속도가 1 mm/s씩 증가
```

### 쉬운 비유

| 가속도 | 의미 | 느낌 |
|--------|------|------|
| 10 mm/s² | 1초 후 10 mm/s 도달 | 매우 느리게 가속 (에스컬레이터) |
| 100 mm/s² | 1초 후 100 mm/s 도달 | 보통 (일반 컨베이어) |
| 1000 mm/s² | 1초 후 1000 mm/s 도달 | 빠르게 가속 (산업 로봇) |

### 가속도가 낮으면 어떻게 되나?

목표 속도에 도달하기 전에 이미 목적지에 가까워져서, 결과적으로 **최고속에 도달하지 못하고 감속**이 시작됩니다.

```
예시: 목표 속도 300 mm/s, 가속도 10 mm/s², 이동 거리 100 mm

최고속 도달까지 필요한 거리 = (300)² ÷ (2 × 10) = 4500 mm
→ 100 mm 이동에서는 최고속에 전혀 도달 못함 → 체감 속도가 매우 느림
```

### RPM/s → mm/s² 변환 (YAML의 default_accel_rpm_s)

YAML의 `default_accel_rpm_s`는 RPM/s 단위입니다.

```
가속도 [mm/s²] = RPM/s × mm/rev ÷ 60
```

**예시 (X축, 300 RPM/s):**
```
300 × 11.999 ÷ 60 = 60 mm/s²
```

**예시 (Z축, 300 RPM/s):**
```
300 × 5.999 ÷ 60 = 30 mm/s²
```

## 3. motor_params.yaml 파라미터 설명

```yaml
# ──────────────────────────────────────────────
# 기본 속도 / 가감속 (launch 시 초기값)
# ──────────────────────────────────────────────

default_velocity_rpm: 30.0     # [RPM]
# → X축: 30 × 11.999 ÷ 60 ≈ 6 mm/s
# → Z축: 30 × 5.999  ÷ 60 ≈ 3 mm/s
# → action goal에서 velocity_mm_s를 지정하면 이 값은 무시됨

default_accel_rpm_s: 300.0     # [RPM/s]
# → X축: 300 × 11.999 ÷ 60 ≈ 60 mm/s²
# → Z축: 300 × 5.999  ÷ 60 ≈ 30 mm/s²

# ──────────────────────────────────────────────
# 호밍 속도
# ──────────────────────────────────────────────

homing_fast_rpm: 30.0    # [RPM] 리밋 스위치 탐색 속도
# → Z축: 30 × 5.999 ÷ 60 ≈ 3 mm/s

homing_slow_rpm: 1.0     # [RPM] Index 펄스 탐색 속도
# → Z축: 1 × 5.999 ÷ 60 ≈ 0.1 mm/s
```

## 4. 실제 이동 속도 설정 방법

### 방법 A: action goal에서 직접 지정 (매 이동마다)

```bash
ros2 action send_goal /motor/move motor_control_interfaces/action/MoveAxis \
  "{x_target_mm: 100.0, z_target_mm: -50.0, velocity_mm_s: 50.0, force_move: false}"
#                                                               ↑
#                                                    이 값이 최우선 적용됨
```

### 방법 B: YAML의 default_velocity_rpm 수정 (launch 기본값 변경)

```yaml
default_velocity_rpm: 30.0    # 느리게 (6 mm/s X축 기준)
default_velocity_rpm: 100.0   # 보통   (20 mm/s X축 기준)
default_velocity_rpm: 250.0   # 빠르게 (50 mm/s X축 기준)
```

> YAML 수정 후에는 launch 재시작 필요.

| RPM  | mm/s  | 용도 예시 |
|------|-------|----------|

| 30   | 6     | 정밀 작업 |

| 250  | 50    | 빠른 이동 |
| 500  | 100   | 고속 이동 |

### Z축 (mm/rev = 5.999)

| RPM  | mm/s  | 용도 예시 |
|------|-------|----------|
| 10   | 1     | 매우 느린 위치 맞춤 |
| 30   | 3     | 정밀 작업 |
| 100  | 10    | 일반 이동 |
| 300  | 30    | 빠른 이동 |
| 600  | 60    | 고속 이동 |

## 6. 가속도 설정 참고표 (default_accel_rpm_s)

| RPM/s | X축 mm/s² | Z축 mm/s² | 특성 |
|-------|-----------|-----------|------|
| 10    | 2         | 1         | 매우 부드럽지만 느림 |
| 50    | 10        | 5         | 부드러운 가속 |
| 100   | 20        | 10        | 보통 |
| 300   | 60        | 30        | 빠른 가속 (권장값) |
| 600   | 120       | 60        | 공격적 가속 |

> **주의:** 가속도가 너무 높으면 기계 진동, 미끄러짐, 위치 오차가 발생할 수 있습니다.
> 처음에는 낮게 설정하고 점진적으로 높이세요.

## 7. 빠른 계산기

원하는 mm/s를 입력하면 YAML에 넣을 RPM을 계산합니다.

```
X축 RPM = 원하는 mm/s × 60 ÷ 11.999
Z축 RPM = 원하는 mm/s × 60 ÷ 5.999

예시: X축 30 mm/s 원할 때
→ 30 × 60 ÷ 11.999 = 150 RPM
→ motor_params.yaml: default_velocity_rpm: 150.0

예시: Z축 20 mm/s 원할 때
→ 20 × 60 ÷ 5.999 = 200 RPM
→ motor_params.yaml: default_velocity_rpm: 200.0
```
